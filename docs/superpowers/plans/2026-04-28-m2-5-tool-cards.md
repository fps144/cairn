**Goal:** 把 M2.4 的"平铺事件行"升级为 spec §6.4 要求的"**可折叠工具卡片流**":`tool_use` + `tool_result` 配对合并成一张卡片(默认折叠,点 ▸ 展开看详情);连续同类工具合并("Read × 3",展开看每条);`assistant_thinking` 默认折叠;`compact_boundary` 渲染成 divider;加 ⌘⇧E 快捷键 toggle 全部;auto-scroll 检测用户手动滚(上滚暂停自动滚)。**不做**:Tab↔Session 绑定(M2.6),"⋮ live" 活跃指示(M2.6),session 选择器(M2.6)。

**Architecture:**
- **数据聚合层**:`TimelineAggregator`(CairnServices,**纯函数**)把 `[Event]` 按规则折成 `[TimelineEntry]`:

  - `.toolCard(use: Event, result: Event?)` —— 配对工具卡(result 可能为 nil,表示 in-flight)
  - `.mergedTools(category: ToolCategory, events: [Event])` —— 连续同 category 合并(N ≥ 2 才合并)
- `TimelineViewModel` 加 `entries: [TimelineEntry]`(每次 events 变动重算)+ `expandedIds: Set<UUID>` 折叠状态
- **UI 层**:`TimelineView` 渲染 `entries`,按 case 分派到 `EventRowView` / `ToolCardView` / `MergedToolsView` / `CompactBoundaryView` / `ThinkingRowView`
- **快捷键**:`⌘⇧E` 在 `CairnApp.commands` 里接,调 vm toggle
- **auto-scroll**:`TimelineView` 用 `@State var userScrolledUp: Bool`,检测 `ScrollView` 内容变化后用户是否在底

**Tech Stack:**
- 纯 Swift 聚合算法(无外部依赖)
- SwiftUI `DisclosureGroup` 或自定义 ▸ 展开(DisclosureGroup 样式 macOS 原生)
- SwiftUI `.keyboardShortcut` 通过 `Commands`
- ScrollView geometry 检测(`ScrollViewReader` + `onChange(of: events.count)` 已有)

## 设计决策(pinned)

| # | 决策 | 理由 |
|---|---|---|
| 1 | `TimelineEntry` **枚举** 代替 "Event 数组 + UI 判断"模式 | 聚合逻辑集中在 aggregator,UI 只负责 case 分派渲染;易测,UI 不含业务 |
| 2 | `TimelineAggregator` **纯函数**`aggregate(events: [Event]) -> [TimelineEntry]` | 无状态;每次 events 变动全量重算;输入 N 行小,性能 OK(N ≈ session event 数,百级) |
| 3 | `.toolCard` 的 result 字段是 **optional** —— tool_use 到了 tool_result 没到时也显示(in-flight 态) | spec §4.4 "重启:从 DB 重建未配对 inflight";UI 显示 "running..." 状态 |
| 4 | `.mergedTools` 仅当 **连续 N ≥ 2 个同 category** 合并;合并前后事件单独 case | 避免"连续两个不同 category 也合"的误解 |
| 5 | 合并粒度是 **category**(不是 toolName):`fileRead × 3` = Read + NotebookRead + Read 也合,`fileWrite × 2` = Write + Edit 也合 | spec §6.4 视觉语言按 category,M0.1 probe 多种 tool 映射到同 category |
| 6 | 不合并 `toolCard` 本身:合并只对"裸 tool_use"(没 tool_result 配对或 result 也被合并)适用;**有 result 配对的总是单独 `.toolCard`** | 配对是 spec §4.4 首要价值,不应被合并掩盖 |
| 7 | `assistant_thinking` 默认折叠(key:`event.id` 存 expandedIds),需点展开 | spec §6.4 明示"灰,折叠" |

| 16 | **不**做 api_usage 折叠/合并(spec "绿悄悄",单行已够小)| 保持简单;Budget 聚合留 M3.x |
| 17 | Event 依赖 M2.4 已有的 `EventStyleMap.symbol/tint`,不重做图标 | 复用 |

## `TimelineEntry` 定义(T1 固化)

```swift
// Sources/CairnServices/TimelineEntry.swift
import Foundation
import CairnCore

public enum TimelineEntry: Equatable, Identifiable {
    case single(Event)
    /// tool_use + 可选 tool_result 的配对卡片。in-flight 时 result == nil。
    case toolCard(toolUse: Event, toolResult: Event?)
    /// 连续同 category 的 tool_use 合并(N ≥ 2)。
    /// - category:合并依据;events.count:显示 "N × category"
    case mergedTools(category: ToolCategory, events: [Event])
    /// compact_boundary 独立渲染为 divider
    case compactBoundary(Event)

    public var id: UUID {
        switch self {
        case .single(let e): return e.id
        case .toolCard(let use, _): return use.id
        case .mergedTools(_, let events): return events.first?.id ?? UUID()
        case .compactBoundary(let e): return e.id
        }
    }
}
```

## `TimelineAggregator` 算法(T1 固化)

```swift
// Sources/CairnServices/TimelineAggregator.swift
import Foundation
import CairnCore

public enum TimelineAggregator {
    public static func aggregate(events: [Event]) -> [TimelineEntry] {
        var entries: [TimelineEntry] = []
        var i = 0

        // 先建 toolUseId → tool_result 的索引,便于配对
        var resultByUseId: [String: Event] = [:]
        for e in events where e.type == .toolResult {
            if let tid = e.toolUseId {
                resultByUseId[tid] = e
            }
        }
        // 已被配对用掉的 tool_result id 集合,避免重复独立渲染
        var consumedResultIds: Set<UUID> = []

        while i < events.count {
            let e = events[i]

            // 已被配对使用的 tool_result 跳过
            if e.type == .toolResult && consumedResultIds.contains(e.id) {
                i += 1
                continue
            }

            switch e.type {
            case .compactBoundary:
                entries.append(.compactBoundary(e))
                i += 1

            case .toolUse:
                // 尝试配对
                if let tid = e.toolUseId, let result = resultByUseId[tid] {
                    entries.append(.toolCard(toolUse: e, toolResult: result))
                    consumedResultIds.insert(result.id)
                    i += 1
                } else {
                    // 未配对 tool_use:尝试与后续同 category 合并(若连续多个未配对同 cat)
                    guard let currentCat = e.category else {
                        entries.append(.toolCard(toolUse: e, toolResult: nil))
                        i += 1
                        break
                    }
                    var runGroup: [Event] = [e]
                    var j = i + 1
                    while j < events.count {
                        let next = events[j]
                        guard next.type == .toolUse,
                              next.category == currentCat,
                              let tid = next.toolUseId,
                              resultByUseId[tid] == nil  // 同样未配对
                        else { break }
                        runGroup.append(next)
                        j += 1
                    }
                    if runGroup.count >= 2 {
                        entries.append(.mergedTools(category: currentCat, events: runGroup))
                    } else {
                        entries.append(.toolCard(toolUse: e, toolResult: nil))
                    }
                    i = j

                }

            default:
                entries.append(.single(e))
                i += 1
            }
        }
        return entries
    }
}
```

**不变量**:
- `aggregate` 输入顺序 = 输出顺序(line_number, block_index)
- 每个 event 出现在**一个** entry 里,不丢不重
- `.mergedTools.events.count >= 2`

```swift
import XCTest
import CairnCore
@testable import CairnServices

final class TimelineAggregatorTests: XCTestCase {
    private func makeEvent(
        type: EventType, category: ToolCategory? = nil,
        toolUseId: String? = nil, lineNumber: Int64 = 1,
        blockIndex: Int = 0
    ) -> Event {
        Event(
            sessionId: UUID(), type: type, category: category,
            toolName: category.map { _ in "T" }, toolUseId: toolUseId,
            timestamp: Date(), lineNumber: lineNumber, blockIndex: blockIndex,
            summary: "s"

        XCTAssertTrue(TimelineAggregator.aggregate(events: []).isEmpty)
    }

        let e = makeEvent(type: .userMessage)
        let entries = TimelineAggregator.aggregate(events: [e])
        XCTAssertEqual(entries.count, 1)
        if case .single(let got) = entries[0] {
            XCTAssertEqual(got.id, e.id)
        } else { XCTFail() }
    }

    func test_toolUseAndResult_pair() {
        let use = makeEvent(type: .toolUse, category: .shell,

        let result = makeEvent(type: .toolResult, toolUseId: "tu_1",
                               lineNumber: 2)

        XCTAssertEqual(entries.count, 1)
        if case .toolCard(let u, let r) = entries[0] {
            XCTAssertEqual(u.id, use.id)
            XCTAssertEqual(r?.id, result.id)
        } else { XCTFail() }
    }

    func test_toolUseWithoutResult_inflight() {
        let use = makeEvent(type: .toolUse, category: .shell,
                            toolUseId: "tu_1")
        let entries = TimelineAggregator.aggregate(events: [use])
        if case .toolCard(_, let r) = entries[0] {
            XCTAssertNil(r, "in-flight:result 应为 nil")
        } else { XCTFail() }
    }

    func test_threeUnpairedSameCategory_merge() {
        let events = (1...3).map { i in
            makeEvent(type: .toolUse, category: .fileRead,
                      toolUseId: "tu_\(i)", lineNumber: Int64(i))

        let entries = TimelineAggregator.aggregate(events: events)
        XCTAssertEqual(entries.count, 1)
        if case .mergedTools(let cat, let es) = entries[0] {

            XCTAssertEqual(es.count, 3)
        } else { XCTFail() }
    }

    func test_twoDifferentCategories_noMerge() {

        } else { XCTFail() }

            XCTAssertEqual(u.id, use.id)
        } else { XCTFail() }
        if case .single(let e3) = entries[3] {
            XCTAssertEqual(e3.type, .assistantText)
        } else { XCTFail() }
    }

    func test_pairedToolResult_notRepeated() {
        let use = makeEvent(type: .toolUse, category: .shell,
                            toolUseId: "tu_1", lineNumber: 1)

                               lineNumber: 2)
```

```swift
    }

    public func toggle(_ id: UUID) {
        if expandedIds.contains(id) { expandedIds.remove(id) }
        else { expandedIds.insert(id) }
    }

    /// ⌘⇧E:有任一折叠态则展开所有,否则折叠所有。
    public func toggleExpandAll() {
        let allIds = Set(entries.map { $0.id })
        // 默认折叠的 entries(toolCard、assistant_thinking)
        let defaultCollapsedIds = entries.compactMap { entry -> UUID? in
            switch entry {
            case .toolCard(let use, _): return use.id
            case .single(let e) where e.type == .assistantThinking: return e.id
            default: return nil
            }
        }
        // 当前折叠态 = 默认折叠 ∪ (allIds \ expandedIds) —— 复杂,简化为:
        // 若 expandedIds 为 allIds 子集的真子集 → 展开所有;否则清空(全折叠)
        if expandedIds.count < allIds.count {
            expandedIds = allIds
        } else {
            expandedIds.removeAll()
        }
        _ = defaultCollapsedIds  // 防 warning
    }

    /// 判断 entry 是否应展开。UI 用。
    public func isExpanded(_ entry: TimelineEntry) -> Bool {
        let id = entry.id
        if expandedIds.contains(id) { return true }
        // 默认折叠规则
        switch entry {
        case .toolCard: return false
        case .single(let e) where e.type == .assistantThinking: return false
        default: return true
        }
    }
}
```

```swift
struct TimelineView: View {

    @State private var isNearBottom: Bool = true

    var body: some View {
        // ... ScrollViewReader proxy
        ScrollView {
            GeometryReader { scrollGeom in
                Color.clear.preference(
                    key: ScrollOffsetKey.self,

                )
            }
            .frame(height: 0)
            LazyVStack { ... }
        }
        .coordinateSpace(name: "timeline")
        .onPreferenceChange(ScrollOffsetKey.self) { offset in
            // 近底判断:若内容底部在可视区内 ±50pt,则 isNearBottom = true
            // 简化实现:暂不精确,用 events.count 大幅增加但 scroll 未动来推断

    }
}
```
